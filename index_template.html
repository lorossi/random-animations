random animations

by Lorenzo Rossi

This is a collection of random animations and static images I made in JS, using my template/library/crude engine to manipulate HTML5 canvases.

Some of these animations are interactive, some are not: generally, if you don't see anything moving, try clicking on the canvas.

Finally, some of these animations are very cpu intensive, so they might not work on mobile devices and may cause your browser to freeze.

Not all of these animations are finished, and sometimes I might add new features to the existing ones. Click around and find out!
